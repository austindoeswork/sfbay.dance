# SFBAY.DANCE

### dev environment setup

```
# check python version
% python3 --version
Python 3.10.6

# clone repo
% git clone git@github.com:austindoeswork/sfbay.dance.git
% cd sfbay.dance

# setup python env
% python3 -m venv venv
% source venv/bin/activate
% pip install -r requirements.txt
% pip install -e .

```

### run

```
# to run the scraper
% python collect.py
```
